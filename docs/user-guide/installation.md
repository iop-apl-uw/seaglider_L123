# Installation

Clone the repository from GitHub, or copy everything in the directory tree locally.

This project requires Python 3.13. You are strongly advised to use
[uv](https://github.com/astral-sh/uv) to run this code — it manages the Python version and
dependencies for you.

To install the runtime dependencies:

```sh
uv sync
```

See [Development](../development/index.md) for the additional dependency groups needed to run
tests, linting, and build these docs.
