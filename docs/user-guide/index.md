# User Guide

This section covers everything needed to install the software, configure it for a
particular Seaglider mission, and run the L2/L3 processing and plotting tools.

1. [Installation](installation.md) — cloning the repo and installing dependencies with `uv`.
2. [Configuration](configuration.md) — the `var_meta.yml` and `mission_meta.yml` config files, and the `processing_config` parameters.
3. [Running L2/L3 Processing](running-processing.md) — the `sg_l123.py` command-line tool.
4. [Plotting](running-plots.md) — the `sg_l123_plot.py` command-line tool.

For background on Seaglider data and the definition of the L1/L2/L3 processing levels, see
[Seaglider_data_L123.pdf](../Seaglider_data_L123.pdf).
